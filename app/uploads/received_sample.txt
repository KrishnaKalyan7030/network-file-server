How Data Moves: The OSI Model Layers 
The OSI Model defines seven distinct functional layers to standardise how data transmits over a network. The four core layers managing data traffic are: 

Zero To Mastery
 +1
Layer 1 (Physical): Governs the raw bitstream transmission over physical media like optical fibers, ethernet cables, or Wi-Fi radio frequencies.
Layer 2 (Data Link): Packages raw bits into data frames and ensures accurate delivery using physical MAC addresses embedded in hardware.
Layer 3 (Network): Assigns logical IP addresses and determines the most efficient routing paths to send data packets between separate networks.
Layer 4 (Transport): Uses protocols like TCP to manage connection reliability, error checking, flow control, and proper data packet reassembly.
